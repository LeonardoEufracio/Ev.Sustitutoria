# Backend API
Laravel 12 organiza modelos, middleware, servicio JWT, controladores y rutas en `routes/api.php`. `FleetController` aplica bloqueos transaccionales al programar, iniciar y cerrar servicios. El kilometraje recorrido se calcula siempre en el servidor como `odómetro final - odómetro inicial`; el cliente no puede enviar ese resultado. El inicio genera una salida y cambia la unidad a `En ruta`; el retorno genera su registro y el cierre libera conductor y vehículo. Los endpoints siguen el prefijo `/api` del enunciado. La validación devuelve 422, autenticación 401 y rol inválido 403. Configure `DB_CONNECTION=sqlsrv`, credenciales SQL Server y un `JWT_SECRET` largo en `.env`.
