# Casos de uso
- CU01 Autenticarse: todos los roles ingresan con correo y contraseña.
- CU02 Registrar solicitud: el solicitante registra fecha, hora, origen, destino, pasajeros y motivo.
- CU03 Evaluar solicitud: el coordinador aprueba o rechaza desde su bandeja.
- CU04 Programar servicio: el coordinador asigna vehículo, conductor y horario.
- CU05 Ejecutar ruta: el conductor consulta su hoja y registra kilometrajes.
- CU06 Finalizar servicio: el sistema libera recursos y actualiza estados.
- CU07 Consultar indicadores: roles autorizados visualizan el resumen operativo.
- CU08 Administrar catálogos y auditoría: administrador/coordinador mantienen recursos.
